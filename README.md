### webapp-reader

> 本项目是一个基于H5的移动端webapp阅读器，具有背景切换、字体大小调节、分页、轻点屏幕唤出上下边栏菜单等功能。采用SCSS实现页面样式、CSS3实现圆角的效果，针对页面上的小图标使用base64格式的图片减少HTTP请求，base64插件对请求数据进行解码。

**技术栈**

- HTML5
- CSS3
- JavaScript
- Gulp
